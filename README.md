#  Sistem Perhitungan Gaji Pegawai (OOP Java)

##  Deskripsi Proyek

Project ini merupakan implementasi Pemrograman Berorientasi Objek (OOP)
menggunakan Java untuk menghitung total gaji pegawai berdasarkan: -
Jabatan pegawai - Presensi jam masuk dan keluar - Jam lembur - Tunjangan
keluarga

Konsep OOP yang diterapkan: - Class & Abstract Class - Interface -
Inheritance - Polymorphism - Package

##  Struktur Folder

projectGaji/ 

├── model/ 

├── service/ 

└── main/

##  Konsep Utama

Interface digunakan sebagai kontrak perhitungan gaji. Abstract class
sebagai superclass pegawai. Subclass mengimplementasikan polimorfisme.

##  Cara Menjalankan

javac main/MainApp.java java main.MainApp

##  Contoh Output

Gaji Supervisor : 11510000

Gaji Teknisi    : 3300000

Gaji HRD        : 5905000

Gaji Karyawan   : 2010000
